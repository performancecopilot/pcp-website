Download

Pre-built Performance Co-Pilot packages and containers can be installed from Packagecloud.

Please follow the installation instructions on how to setup the repositories for your distribution.

Already Included

Performance Co-Pilot is already included in all popular distributions (RHEL, Debian, Fedora, SUSE, Ubuntu). Use the commands below to install from your distribution.

Installation by Platform

Debian / Ubuntu

apt-get install pcp
invoke-rc.d pmcd start
invoke-rc.d pmie start
invoke-rc.d pmlogger start

Fedora / RHEL

yum install pcp
# RHEL 6
service pmcd start
service pmlogger start
service pmie start
# RHEL 7 and later / Fedora
systemctl start pmcd
systemctl start pmie
systemctl start pmlogger

macOS

brew tap performancecopilot/performancecopilot
brew install --cask pcp-perf

Alternatively, download the DMG installer from GitHub releases.

openSUSE

Direct Install link

Solaris

No pre-built packages. Build from source (see below).

Windows

No pre-built packages. Build from source (see below).
